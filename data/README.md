## Dataset Information

This directory contains the agricultural production dataset for India covering 24 years of agricultural data across multiple dimensions.

### Dataset Overview

| Attribute | Details |
|-----------|---------|
| **Time Span** | 1997-98 to 2020-21 (24 years) |
| **Total Records** | 345,407 |
| **Number of Crops** | 56 |
| **Seasons** | 5 (Whole Year, Monsoon, Winter, Autumn, Summer) |
| **Geographic Coverage** | All Indian states and districts |
| **Data Quality** | No missing values or duplicates |

### File Information
- **Filename**: `India Agriculture Crop Production.csv`
- **Format**: CSV (Comma-Separated Values)
- **Columns**: State, District, Year, Season, Crop, Production, Yield

### Dataset Columns

| Column | Description | Type |
|--------|-------------|------|
| **State** | Indian state/territory name | String |
| **District** | District within the state | String |
| **Year** | Agricultural year (e.g., 1997-98) | String |
| **Season** | Crop season | String |
| **Crop** | Name of the crop | String |
| **Production** | Production in units | Numeric |
| **Yield** | Yield in units/hectare | Numeric |
