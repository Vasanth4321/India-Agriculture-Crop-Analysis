### Production Statistics
- **Average Production**: 944,528 units per record
- **Median Production**: 717 units
- **Average Yield**: 79.40 units
- **Production Range**: Highly variable (driven by large-scale producers)

### Trends & Patterns
- Production exhibits **upward trajectory** from 1997-98 to 2019-20
- **Yearly fluctuations** observed (climate & policy impacts)
- 2020-21 data incomplete (319 records)
- **Seasonal Dominance**: Whole Year > Kharif > Rabi
